Comando e controllo on-premise

Progetto gestione picchetti per Polizia Locale di Padova

Costruito per Polizia Locale di Padova, questa piattaforma web on-premise gestisce turni di servizio operativo e reporting con validazione rigorosa e controllo di approvazione gerarchico.

  • Distribuzione on-premise su cluster di server Linux, non cloud pubblico
  • Backend Laravel che alimenta logica di servizio modulare e flussi di lavoro sicuri per ruoli
  • Orchestrazione Kubernetes per resilienza dei container e separazione degli ambienti
  • Controlli di accesso granulari e stati di approvazione immutabili per affidabilità operativa

Backend
